§ STRINGENCODING

enum STRINGENCODING
Enumerator
STRINGENCODING_XBIT 

An OS dependent encoding. Only use this if passing data to OS functions. The actual coding is unspecified.

STRINGENCODING_8BIT 

8-bit ASCII. Discards Unicode information.

STRINGENCODING_7BIT 

7-bit ASCII. Does not understand any international characters, e.g. "åäüö".

STRINGENCODING_7BITHEX 

7-bit ASCII, with any non 7-bit characters encoded in the text as e.g. "\uEFA0".

STRINGENCODING_UTF8 

UTF-8.

STRINGENCODING_HTML 

HTML encoding.
