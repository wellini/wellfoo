§ FOREACH

#define FOREACH (   it,
  iterable 
)

Use this macro to iterate over an iterable object. If the type of the iterable object has a member type named ForEachIteratorMarker, then it is expected that the iterable object is a foreach-iterator itself, supporting the foreach-protocol consisting of operator ++() and operator Bool(). Otherwise, an AutoIterator is created for the object.

Parameters
[in]itThe iterator variable to use.
[in]iterableThe object to iterate over.
