§ SWITCH_FALLTHROUGH

#define SWITCH_FALLTHROUGH

This statement needs to be used for any case in a switch the contains code, but does not call break. An example:

switch(condition)
{
case 0:
...do something...
// SWITCH_FALLTHROUGH
case 1:
...do something...
break;
}

§ DISALLOW_COPY_AND_ASSIGN

#define DISALLOW_COPY_AND_ASSIGN (   TypeName)

This statement needs to be used at the beginning of any class that cannot be copied An example:

class MyClass
{
DISALLOW_COPY_AND_ASSIGN(MyClass);
...
};

§ MOVE_ASSIGNMENT_OPERATOR

#define MOVE_ASSIGNMENT_OPERATOR (   TypeName)

This statement automatically creates a move assignment operator from an existing move constructor. It needs to be written in the public: section. An example:

class MyClass
{
DISALLOW_COPY_AND_ASSIGN(MyClass);
public:
MOVE_ASSIGNMENT_OPERATOR(MyClass);
...
};
