Detailed Description

Represents Cinema 4D's polygon object.

Note
Has to be created with Alloc() and destroyed with Free().

§ CreatePhongNormals()

Vector32* CreatePhongNormals ( )

Gets a buffer with the stored phong normals for the polygon object.
Buffer contains 4*GetPolygonCount() vectors, one for each point of each face (including the unused fourth point of a triangle). The buffer has to be freed afterwards with DeleteMem().

Note
This method is only valid if the object has a phong tag.
Returns
A pointer to the buffer with phong normals.
