Changes in Melange 15.0

Table of Contents

  • New Features
  • Naming Changes
  • API Changes

New Features

  • Melange version number now follows the Cinema 4D file version it reads/writes.
    Melange 15.0 reads Cinema 4D file format up to R15 and writes Cinema 4D R15 file format (version 15050).

Naming Changes

Melange 15.0 brings important naming changes. See Name Changes in Melange 15.0 for more information to adapt existing code.

API Changes

  • Renamed _melange_ namespace to melange
  • Refactored math types, classes and functions : vector, matrices, quaternions, c4d_tools.h etc.
  • BaseList2D::GetLayer/BaseList2D::SetLayer renamed to GetLayerObject/SetLayerObject
  • State struct renamed to AxisState (see BaseDocument::GetState).
  • Added new IP communication command RENDERCONNECTION_COMMAND_RESETRENDERSERVER (see IpCommunicationThread::ResetRenderServer).
  • Add new send texture command RENDERCONNECTION_COMMAND_SENDTEXTURE with flag to save to memory only (see IpCommunicationThread::SendTexture).
  • Removed stream scene command RENDERCONNECTION_COMMAND_STREAMSCENE.
  • Added new knot flags FLAG_KNOT.
  • Removed deprecated knot flags SPLINE_KNOT_FLAG_NO_HORIZ_MOVE and SPLINE_KNOT_FLAG_NO_VERT_MOVE.
  • Added ConstUVWHandle. Adapted UVWTag::GetDataAddressR and UVWTag::Get. Added UVWTag::Copy.
  • Adapted NormalTag. Renamed GetNormalCount to NormalTag::GetDataCount.
  • Removed GetNormals/SetNormals. Added NormalTag::GetDataAddressR/NormalTag::GetDataAddressW, NormalTag::Get/NormalTag::Set and NormalTag::Copy.
  • Added NormalHandle and ConstNormalHandle.
  • Renamed GeGetCpuCount to GeGetCurrentThreadCount and changed return type to Int32 instead of int.
